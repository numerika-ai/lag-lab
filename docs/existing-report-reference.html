7. Metodologia

Korelacja

Spearman rank cross-correlation - odporna na outliers, mierzy monotoniczne związki

Okno Lag

±120s dla shitcoinów, ±60s dla majors. Adaptive window based on volatility

Dane Live

bookTicker (best bid/ask) z WebSocket, najwyższa częstotliwość aktualizacji

Dane Historyczne

aggTrades z parquets, 70h danych na Spark, microsecond precision

Minimum Thresholds

Min. 30 wspólnych sekund, 15 consecutive returns dla stabilności

Statistical Robust

Handles missing data, outliers, different sampling rates gracefully
