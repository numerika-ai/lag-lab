// Methodology

Spearman Rank

Cross-correlation using Spearman rank — robust to outliers, measures monotonic relationships

Multi-Timeframe

Sub-windows: 30s, 60s, 120s, 300s. Boundary artifact filter at 0.9× window size

Forward-Fill

Sparse tick data forward-filled to continuous 1s series for consistent correlation measurement

WebSocket bookTicker

Best bid/ask real-time feed. Highest frequency updates. Mid-price to 1s returns.

Infrastructure
